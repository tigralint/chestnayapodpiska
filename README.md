# Честная Подписка

Бесплатный сервис для борьбы с ненужными списаниями и темными паттернами. Генерируем юридически грамотные претензии, обучаем цифровой самообороне и подсвечиваем наглые сервисы на «Народном радаре».

## 🚀 Основные возможности

*   **Умный конструктор претензий** — формирует документ на возврат денег за подписки или онлайн-курсы (ст. 32 ЗоЗПП, ст. 782 ГК РФ).
*   **Гибкий выбор тона** — от вежливого обращения лояльного клиента до жесткой досудебной претензии с упоминанием Роспотребнадзора.
*   **Тренажер отписки** — интерактивное обучение поиску тех самых «спрятанных» кнопок отмены.
*   **Народный радар** — карта сервисов, которые чаще всего списывают деньги в тихую.
*   **VisionOS интерфейс** — современный дизайн с эффектом стекла, размытием и живыми анимациями.
*   **Экспорт в Word** — скачивание готового файла `.docx` для печати или отправки по почте.

## 🛠 Технический стек

*   **Фреймворк**: React + Vite + TypeScript.
*   **Стили и дизайн**: Tailwind CSS (кастомная дизайн-система в стиле Glassmorphism).
*   **Серверная часть**: Netlify Functions (Node.js).
*   **Интеллект**: OpenRouter (Gemini / Qwen) для генерации текстов претензий.
*   **Безопасность**: Cloudflare Turnstile для защиты от ботов.

## � Как запустить проект

### 1. Подготовка
Тебе понадобятся:
- Node.js 18+
- Ключ от OpenRouter (для работы ИИ)

### 2. Установка
```bash
git clone https://github.com/your-username/podpiska.git
cd podpiska
npm install
```

### 3. Переменные окружения
Создай файл `.env` в корне проекта и добавь:
```env
OPENROUTER_API_KEY=твой_ключ_здесь
VITE_TURNSTILE_SITE_KEY=ключ_сайта_turnstile
TURNSTILE_SECRET_KEY=секретный_ключ_turnstile
```

### 4. Запуск
Для полной работы (с функциями бэкенда) лучше использовать Netlify CLI:
```bash
npm install -g netlify-cli
netlify dev
```
